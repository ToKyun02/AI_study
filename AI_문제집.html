Q 01
AI 계층 구조

AI · ML · DL의 포함 관계를 올바르게 나타낸 것은?

  • ① DL ⊃ ML ⊃ AI
  • ② ML ⊃ AI ⊃ DL
  • ③ AI ⊃ ML ⊃ DL
  • ④ AI ⊃ DL ⊃ ML
정답: ③ — AI가 가장 넓은 범주이며, ML이 AI의 하위, DL이 ML의 하위 분야입니다.
Q 02
ANN 발전

단일 퍼셉트론(Single-layer Perceptron)의 근본적인 한계로 옳은 것은?

  • ① 이진 분류를 처리할 수 없다
  • ② XOR과 같은 비선형 문제를 해결할 수 없다
  • ③ 가중치 학습이 불가능하다
  • ④ 입력 데이터가 2개 이상이면 동작하지 않는다
정답: ② — 퍼셉트론은 선형 결정 경계만 표현 가능하므로 XOR 같은 비선형 문제를 해결할 수 없습니다. 이를 해결하기 위해 MLP(다층 퍼셉트론)가 등장했습니다.
Q 03
활성화 함수

현재 딥러닝에서 은닉층의 기본 활성화 함수로 가장 많이 사용되며, 기울기 소실 문제를 크게 완화한 함수는?

  • ① Sigmoid
  • ② tanh
  • ③ ReLU
  • ④ Softmax
정답: ③ — ReLU(Rectified Linear Unit)는 양의 입력에 대해 기울기가 1로 유지되어 기울기 소실을 크게 줄입니다. AlexNet에서 도입 후 딥러닝의 표준 활성화 함수가 되었습니다.
Q 04
옵티마이저

Momentum 기법과 RMSProp의 적응 학습률을 결합하여 딥러닝의 사실상 기본 옵티마이저가 된 알고리즘은?

  • ① GD (Gradient Descent)
  • ② Adagrad
  • ③ Adam
  • ④ AdaDelta
정답: ③ — Adam(Adaptive Moment Estimation)은 1차 모멘트(Momentum)와 2차 모멘트(RMSProp)를 모두 활용하여 대부분의 태스크에서 빠르고 안정적으로 수렴합니다.
Q 05
규제·과적합

L1 규제(Lasso)의 특성으로 옳은 것은?

  • ① 가중치를 0에 가깝게 축소하지만 정확히 0으로 만들지 않는다
  • ② 불필요한 특성의 가중치를 정확히 0으로 만들어 특성 선택 효과를 낸다
  • ③ 학습 중 랜덤으로 뉴런을 비활성화한다
  • ④ 원본 데이터를 변환하여 학습 데이터 다양성을 높인다
정답: ② — L1 규제는 가중치 절댓값 합을 패널티로 추가하여 희소 모델(sparse model)을 만들고 자동 특성 선택 효과를 냅니다. ①은 L2 규제, ③은 Dropout, ④는 데이터 증강의 설명입니다.
Q 06
CNN

CNN에서 커널(Kernel)의 가중치 공유(Weight Sharing)가 가져오는 핵심 이점은?

  • ① 역전파 알고리즘을 사용하지 않아도 된다
  • ② 파라미터 수를 대폭 절감하고 위치 불변성을 학습한다
  • ③ 활성화 함수 없이도 비선형 특징을 학습한다
  • ④ 시계열 데이터를 처리할 수 있게 된다
정답: ② — 하나의 커널이 입력 전체를 슬라이딩하며 같은 가중치를 적용하므로 파라미터가 크게 줄고, 이미지 어디서든 동일 패턴을 인식하는 위치 불변성(Translation Invariance)을 얻습니다.
Q 07
이미지 모델 변천사

2015년 등장하여 잔차 연결(Skip Connection)으로 100층 이상의 매우 깊은 신경망 학습을 가능하게 한 모델은?

  • ① AlexNet
  • ② VGGNet
  • ③ ResNet
  • ④ ViT
정답: ③ — ResNet은 잔차 블록(Residual Block)의 Skip Connection으로 기울기 소실 문제를 근본적으로 해결하여 현재도 다양한 태스크의 백본으로 광범위하게 사용됩니다.
Q 08
RNN & LSTM

LSTM이 기존 RNN에 추가한 핵심 구성 요소로, 장기 기억 경로 역할을 하는 것은?

  • ① Attention Mechanism
  • ② Cell State
  • ③ Positional Encoding
  • ④ Residual Connection
정답: ② — LSTM은 Cell State라는 장기 기억 전용 경로를 추가하고, Forget / Input / Output 게이트 3개로 정보 흐름을 제어하여 RNN의 장기 의존성 문제를 크게 완화합니다.
Q 09
머신러닝 학습 종류

정답 레이블(Label) 없이 데이터의 숨겨진 구조와 패턴을 스스로 학습하는 방식은?

  • ① 지도 학습 (Supervised Learning)
  • ② 강화 학습 (Reinforcement Learning)
  • ③ 비지도 학습 (Unsupervised Learning)
  • ④ 전이 학습 (Transfer Learning)
정답: ③ — 비지도 학습은 레이블 없이 데이터 분포·구조를 학습합니다. K-means 군집화, PCA 차원 축소 등이 대표적 알고리즘입니다.
Q 10
학습 파이프라인

신경망 학습의 올바른 파이프라인 순서는?

  • ① 역전파 → 순전파 → 손실 계산 → 가중치 업데이트
  • ② 손실 계산 → 순전파 → 역전파 → 가중치 업데이트
  • ③ 순전파 → 손실 계산 → 역전파 → 가중치 업데이트
  • ④ 순전파 → 역전파 → 가중치 업데이트 → 손실 계산
정답: ③ — 데이터를 순전파하여 예측값을 얻고 → 손실 함수로 오차를 계산한 뒤 → 역전파로 각 가중치의 기울기를 구하고 → 옵티마이저로 가중치를 업데이트합니다.
Q 11
데이터 전처리

범주형 데이터를 숫자로 변환할 때, 순서 관계가 없는 범주에 가장 적합한 인코딩 방법은?

  • ① Label Encoding
  • ② One-Hot Encoding
  • ③ Min-Max 정규화
  • ④ Z-score 표준화
정답: ② — One-Hot Encoding은 범주를 이진 벡터로 변환하여 순서 관계를 부여하지 않습니다. Label Encoding은 정수를 부여하므로 모델이 순서 관계가 있다고 오해할 수 있습니다.
Q 12
생성 모델

GAN(생성적 적대 신경망)의 구성 요소로 올바르게 짝지어진 것은?

  • ① 인코더 — 디코더
  • ② 생성자(Generator) — 판별자(Discriminator)
  • ③ Teacher 모델 — Student 모델
  • ④ 순전파망 — 역전파망
정답: ② — GAN에서 생성자는 가짜 데이터를 만들고, 판별자는 진짜/가짜를 구별하는 역할을 합니다. 이 두 구성 요소가 적대적으로 경쟁하며 함께 학습됩니다.
Q 13
파운데이션 모델

Diffusion Model에서 LDM(Latent Diffusion Model)으로 발전한 핵심 이점은?

  • ① 생성자와 판별자 구조를 도입하여 품질을 높였다
  • ② 픽셀 공간 대신 잠재 공간에서 확산을 수행해 연산량을 대폭 절감했다
  • ③ 텍스트 없이 이미지만으로 멀티모달 학습을 가능하게 했다
  • ④ 순환 신경망 구조를 결합하여 시계열 이미지를 생성했다
정답: ② — LDM은 인코더로 이미지를 잠재 공간으로 압축한 뒤 그 공간에서 Diffusion을 수행하여, 픽셀 공간에서의 연산 대비 학습·추론 속도를 수십 배 향상시킵니다.
Q 14
학습 고려사항

모델이 훈련 데이터에는 매우 높은 성능을 보이지만 새로운 데이터에서는 성능이 크게 떨어지는 현상은?

  • ① Underfitting (과소적합)
  • ② Distribution Shift
  • ③ Overfitting (과적합)
  • ④ Gradient Explosion
정답: ③ — 과적합(Overfitting)은 모델이 훈련 데이터의 노이즈까지 외워버려 일반화 성능이 낮아지는 현상입니다. L1/L2 규제, Dropout, 데이터 증강, 조기 종료로 방지합니다.
Q 15
전이 학습

사전학습된 모델의 일부 레이어를 동결(Freeze)하고 상위 레이어만 새 데이터로 재학습하는 전이 학습 방식은?

  • ① Zero-shot Learning
  • ② Fine-tuning
  • ③ Knowledge Distillation
  • ④ Pre-training from Scratch
정답: ② — Fine-tuning은 사전학습된 파운데이션 모델의 가중치를 초기값으로 사용하고 특정 태스크 데이터로 추가 학습하는 방식으로, 현재 AI 개발의 표준 패러다임입니다.

Q 28
ANN 발전 · 2점

퍼셉트론(Perceptron)에서 MLP(다층 퍼셉트론), 그리고 DNN(심층 신경망)으로 발전하게 된 이유를 각 단계별 한계와 해결 방법을 포함하여 서술하시오.

✦ 서술 포인트: ① 퍼셉트론의 한계(XOR, 비선형) → ② MLP의 등장(역전파, 은닉층) → ③ MLP의 한계(기울기 소실) → ④ DNN의 해결(깊이, ReLU 등)
퍼셉트론은 입력에 가중치를 곱하고 임계값을 비교하는 단순한 이진 분류 장치로, 단일 레이어의 선형 결정 경계만 표현 가능하여 XOR과 같은 비선형 문제를 해결할 수 없다는 근본적 한계가 있었다.

이를 해결하기 위해 여러 퍼셉트론을 층층이 쌓아 은닉층을 구성한 MLP(다층 퍼셉트론)가 등장했다. 역전파(Backpropagation) 알고리즘을 통해 가중치를 학습할 수 있게 되어 비선형 문제 해결이 가능해졌다. 그러나 레이어가 깊어질수록 Sigmoid 등의 활성화 함수를 통과할 때 기울기가 소실되어 학습이 어려워지는 한계가 있었다.

이에 따라 DNN(심층 신경망)은 ReLU 활성화 함수 도입, 배치 정규화, 적절한 초기화 기법 등을 통해 기울기 소실 문제를 완화하고 수십~수백 개의 은닉층을 학습할 수 있게 되었다. 이를 통해 이미지·음성·자연어 처리에서 인간 수준 또는 초월 성능을 달성했다.
Q 29
CNN 구조 · 2점

CNN(합성곱 신경망)의 입력부터 출력까지의 전체 처리 흐름을 주요 구성 요소와 각각의 역할을 포함하여 서술하시오.

✦ 서술 포인트: 입력 텐서 → 패딩 → 합성곱(커널) → 활성화(ReLU) → 특징맵 → 풀링 → Flatten → FC Layer → Softmax 출력
CNN은 다음 순서로 이미지를 처리한다. ① 입력: 이미지는 (배치, 높이, 너비, 채널)의 4D 텐서로 입력된다. ② 패딩: 입력 주변에 0을 추가하여 합성곱 후 공간 크기를 보존하거나 경계 정보 손실을 방지한다. ③ 합성곱 연산: 커널(필터)이 입력 위를 슬라이딩하며 내적 연산을 수행, 가중치를 공유하여 파라미터를 절감하고 위치 불변성을 학습한다. ④ 활성화 함수(ReLU): 비선형성을 부여한다. ⑤ 특징맵: 각 필터가 반응하는 패턴(엣지, 텍스처 등)이 출력되며, 레이어가 깊을수록 고수준 특징이 추출된다. ⑥ 풀링: 공간 크기를 줄여 계산량을 감소시킨다(Max Pooling이 주로 사용). ⑦ Flatten: 다차원 특징맵을 1D 벡터로 변환한다. ⑧ FC Layer: 모든 특징을 결합하여 클래스별 점수를 계산한다. ⑨ Softmax: 점수를 확률 분포로 변환하여 최종 분류 결과를 출력한다.
Q 30
옵티마이저 비교 · 2점

GD(경사하강법), SGD(확률적 경사하강법), Adam의 차이점을 비교하고, 실무에서 Adam이 기본값으로 선호되는 이유를 서술하시오.

✦ 서술 포인트: GD(전체 데이터, 안정적이지만 느림) vs SGD(미니배치, 빠르지만 불안정) → Adam(Momentum+RMSProp, 적응 학습률, 대부분 태스크 효과적)
GD(경사하강법)은 전체 데이터셋의 손실을 계산한 뒤 가중치를 업데이트하므로 수렴이 안정적이지만, 대규모 데이터에서 한 번의 업데이트에 막대한 시간과 메모리가 소요된다는 단점이 있다.

SGD(확률적 경사하강법)은 미니배치 단위로 기울기를 계산하여 업데이트 속도가 매우 빠르고 노이즈가 local minimum 탈출에 도움을 주지만, 수렴 경로가 불안정하고 학습률 설정에 민감하다는 단점이 있다.

Adam은 Momentum(이전 기울기 방향의 관성 누적)과 RMSProp(파라미터별 적응 학습률)을 결합한 알고리즘이다. 각 파라미터마다 학습률을 자동으로 조정하여 희소 기울기와 노이즈 문제를 동시에 처리하며, 대부분의 태스크에서 빠르고 안정적으로 수렴한다. 학습률 튜닝 부담도 적어 실무에서 기본 옵티마이저로 사용된다.
Q 31
RNN vs LSTM · 2점

RNN의 장기 의존성(Long-Term Dependency) 문제가 무엇인지 설명하고, LSTM이 이를 어떻게 해결하는지 게이트 구조를 중심으로 서술하시오.

✦ 서술 포인트: RNN의 장기 의존성 문제(기울기 소실, 긴 시퀀스에서 초반 기억 소실) → LSTM의 Cell State + 3개 게이트(Forget/Input/Output)로 해결
RNN은 이전 타임스텝의 hidden state를 현재 입력과 함께 처리하여 시계열 데이터의 "기억"을 구현한다. 그러나 시퀀스가 길어질수록 역전파 시 기울기가 레이어를 거칠수록 점점 작아지는 기울기 소실 문제가 발생하여, 긴 시퀀스의 초반 입력 정보가 점점 사라지는 장기 의존성 문제가 생긴다.

LSTM은 기존 hidden state와 별도로 Cell State라는 장기 기억 전용 경로를 추가하고 세 가지 게이트로 정보 흐름을 정밀하게 제어한다: Forget Gate는 sigmoid를 통해 과거 Cell State를 얼마나 버릴지 결정하고, Input Gate는 현재 정보를 얼마나 새로 기억할지 결정하며, Output Gate는 다음 hidden state로 얼마나 넘길지 결정한다. Cell State를 통해 정보가 비교적 손실 없이 멀리까지 전달될 수 있어 장기 의존성 문제를 크게 완화한다.
Q 32
지도·비지도·강화학습 · 2점

지도 학습, 비지도 학습, 강화 학습의 핵심 차이를 데이터·학습 방식·대표 알고리즘을 포함하여 비교 서술하시오.

✦ 서술 포인트: 지도(레이블 있음, 분류/회귀) / 비지도(레이블 없음, 군집화/차원축소) / 강화(환경-보상, 에이전트 정책 학습)
지도 학습은 입력-출력 쌍의 레이블이 있는 데이터로 학습하여 새 입력에 대한 출력을 예측한다. 회귀(연속값 예측: 선형 회귀)와 분류(범주 예측: 로지스틱 회귀)가 대표적이며 정확도, F1-Score 등으로 평가한다.

비지도 학습은 정답 레이블 없이 데이터의 숨겨진 구조와 패턴을 스스로 학습한다. K-means, 계층적 군집화로 데이터를 그룹화(군집화)하거나, PCA로 고차원 데이터를 저차원으로 압축(차원 축소)하는 것이 대표적이다.

강화 학습은 에이전트가 환경과 상호작용하며 보상 신호를 최대화하는 행동 정책을 시행착오로 학습한다. 정답 데이터 없이 피드백 신호(보상/패널티)만으로 학습하며, 게임·로봇 제어·자율주행에 적합하다. 최근에는 인간 피드백을 보상으로 활용하는 RLHF로 LLM 정렬에도 핵심적으로 활용된다.
Q 33
이미지 모델 변천사 · 2점

AlexNet(2012) → VGGNet(2014) → ResNet(2015) → ViT(2020)의 발전 흐름에서 각 모델이 해결한 문제와 도입한 핵심 혁신을 서술하시오.

✦ 서술 포인트: AlexNet(GPU+ReLU+Dropout) → VGGNet(3×3 커널 단순화) → ResNet(잔차 연결, 기울기 소실 해결) → ViT(Transformer, Self-Attention, 전역 의존성)
AlexNet(2012): 최초로 GPU를 활용한 딥러닝으로 ImageNet 대회를 압도적으로 우승. ReLU 도입으로 기울기 소실을 완화하고, Dropout으로 과적합을 방지했다. GPU 병렬 학습의 가능성을 증명하여 딥러닝 르네상스를 열었다.

VGGNet(2014): 3×3 소형 커널만 일관되게 사용하여 구조를 단순화하고 16~19층을 쌓았다. 구조가 단순하여 전이 학습의 베이스로 오랜 기간 활용되었으나 파라미터가 138M에 달해 메모리 비효율 문제가 있었다.

ResNet(2015): 잔차 연결(Skip Connection)로 입력을 레이어를 건너뛰어 출력에 직접 더하는 구조를 도입했다. 이로써 기울기 소실 문제를 근본적으로 해결하여 100층 이상의 매우 깊은 네트워크 학습이 가능해졌다. 현재도 다양한 태스크의 백본으로 활용된다.

ViT(2020): 이미지를 패치로 분할하고 Transformer의 Self-Attention으로 처리. CNN 없이 이미지 전체의 전역적 의존성을 파악할 수 있어 대규모 데이터에서 CNN 성능을 초월하고, 언어와 비전 모델의 통합을 가능하게 했다.
Q 34
과적합 방지 · 2점

과적합(Overfitting)이 발생하는 원인을 설명하고, L1 규제, L2 규제, Dropout, 데이터 증강의 각각의 작동 원리와 특징적 차이를 비교하여 서술하시오.

✦ 서술 포인트: 과적합 원인(데이터 부족, 모델 과도한 복잡도) → L1(희소화, 특성 선택) / L2(균형 축소, 미분 가능) / Dropout(랜덤 비활성화, 앙상블) / 데이터 증강(다양성 확보)
과적합은 모델이 학습 데이터의 노이즈와 특수한 패턴까지 외워버려 새로운 데이터에 일반화되지 않는 현상이다. 주 원인은 학습 데이터 부족, 모델 복잡도가 데이터보다 과도하게 높은 경우, 학습 에포크 과다 등이다.

L1 규제(Lasso): 손실 함수에 가중치 절댓값 합(|W|)을 패널티로 추가. 불필요한 가중치를 정확히 0으로 만들어 희소 모델을 생성하고 자동 특성 선택 효과를 낸다. 고차원 데이터에 유리하나, 0에서 미분 불가능하다.

L2 규제(Ridge): 손실 함수에 가중치 제곱 합(W²)을 추가. 모든 가중치를 0에 가깝게 줄이지만 정확히 0으로는 만들지 않아 전체 특성을 균형 있게 유지한다. 어디서든 미분 가능하여 최적화가 안정적이다.

Dropout: 학습 중 무작위로 p% 뉴런을 비활성화하여 특정 뉴런에 대한 의존성을 차단한다. 여러 서브 네트워크의 앙상블 효과로 강건한 표현을 학습한다. 추론 시에는 모든 뉴런을 활성화한다.

데이터 증강: 원본 데이터를 회전, 플립, 크롭 등으로 변환하여 학습 데이터의 다양성을 인위적으로 늘린다. 실제 데이터 없이도 모델 강건성을 향상시킬 수 있어 소규모 데이터셋에 특히 효과적이다.
Q 35
생성 모델 계보 · 2점

이미지 생성 AI의 발전 계보를 GAN에서 Stable Diffusion까지 흐름을 따라 서술하고, CLIP과 지식 증류(Knowledge Distillation)가 생성 AI 발전에 기여한 역할을 함께 설명하시오.

✦ 서술 포인트: GAN → Diffusion(노이즈 제거, 안정적) → LDM(잠재 공간, 속도 향상) → CLIP(텍스트-이미지 연결) → Stable Diffusion(Text-to-Image) → 지식 증류(SANA 등 경량화)
GAN(생성적 적대 신경망)은 생성자와 판별자가 경쟁하며 사실적 이미지를 생성하는 현대 생성 모델의 출발점이었다. 그러나 Mode Collapse 등 학습 불안정 문제가 있었다.

Diffusion Model은 데이터에 점진적으로 노이즈를 추가하고(순방향), 노이즈에서 원본을 복원하는 과정(역방향)을 학습한다. GAN보다 안정적이고 다양한 고품질 샘플을 생성하나 수백 스텝의 느린 샘플링이 단점이었다.

LDM(Latent Diffusion Model)은 픽셀 공간이 아닌 인코더로 압축한 잠재 공간에서 Diffusion을 수행하여 연산량을 대폭 절감하고 학습·추론 속도를 수십 배 향상시켰다.

CLIP은 이미지-텍스트 쌍을 대조 학습으로 공통 잠재 공간에 임베딩하여 언어와 비전 사이의 의미적 다리를 구축했다. Stable Diffusion은 LDM에 CLIP 텍스트 인코더를 결합하여 텍스트 프롬프트로 고품질 이미지를 생성하는 Text-to-Image 시대를 열었다.

지식 증류는 대형 Teacher 모델의 소프트 확률 분포를 소형 Student 모델이 학습하게 하는 기법으로, SANA 같은 초고속 경량 모델이 대형 모델 수준의 품질을 유지하면서 모바일·엣지 디바이스에서도 이미지 생성이 가능하게 했다.

Q 36
손실 함수 · MSE

회귀 문제에서 사용하는 평균 제곱 오차(MSE)의 수식으로 올바른 것은? ($n$: 샘플 수, $y_i$: 실제값, $\hat{y}_i$: 예측값)

  • ①
    $\displaystyle \text{MSE} = \dfrac{1}{n}\sum_{i=1}^{n}|y_i - \hat{y}_i|$
  • ②
    $\displaystyle \text{MSE} = \dfrac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$
  • ③
    $\displaystyle \text{MSE} = -\dfrac{1}{n}\sum_{i=1}^{n} y_i \log \hat{y}_i$
  • ④
    $\displaystyle \text{MSE} = \sqrt{\dfrac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}$
정답: ② — MSE는 예측 오차를 제곱하여 평균합니다. ①은 MAE(절댓값 평균), ③은 Cross-Entropy, ④는 RMSE입니다.
Q 37
활성화 함수 · Sigmoid

Sigmoid 함수의 수식으로 올바른 것은?

  • ①
    $\sigma(x) = \max(0, x)$
  • ②
    $\sigma(x) = \dfrac{e^x - e^{-x}}{e^x + e^{-x}}$
  • ③
    $\sigma(x) = \dfrac{1}{1 + e^{-x}}$
  • ④
    $\sigma(x) = \dfrac{e^{x_i}}{\sum_{j} e^{x_j}}$
정답: ③ — ①은 ReLU, ②는 tanh, ④는 Softmax 함수입니다. Sigmoid는 출력을 $(0, 1)$ 범위로 압축합니다.
Q 38
역전파 · 연쇄 법칙

합성 함수 $L = f(g(x))$에 대해 연쇄 법칙(Chain Rule)을 적용한 미분으로 올바른 것은?

  • ①
    $\dfrac{dL}{dx} = \dfrac{dL}{df} + \dfrac{df}{dg} + \dfrac{dg}{dx}$
  • ②
    $\dfrac{dL}{dx} = \dfrac{dL}{df} \cdot \dfrac{df}{dg} \cdot \dfrac{dg}{dx}$
  • ③
    $\dfrac{dL}{dx} = \dfrac{dL}{df} \div \dfrac{dg}{dx}$
  • ④
    $\dfrac{dL}{dx} = \dfrac{d(f \cdot g)}{dx}$
정답: ② — 연쇄 법칙은 합성 함수의 미분을 각 단계 기울기의 곱으로 분해합니다. 역전파는 이를 레이어별로 반복 적용합니다.
Q 39
CNN · 출력 크기

입력 크기 $W$, 필터 크기 $F$, 패딩 $P$, 스트라이드 $S$일 때 합성곱 출력 크기 공식은?

  • ①
    $\displaystyle O = \left\lfloor \dfrac{W - F + 2P}{S} \right\rfloor + 1$
  • ②
    $\displaystyle O = \dfrac{W + F - 2P}{S} + 1$
  • ③
    $\displaystyle O = \left\lfloor \dfrac{W \cdot F}{P \cdot S} \right\rfloor$
  • ④
    $\displaystyle O = (W - F) \cdot S + 2P + 1$
정답: ① — 예: $W=32, F=3, P=1, S=1$이면 $O = \lfloor(32-3+2)/1\rfloor+1 = 32$ (same padding). 이 공식은 CNN 설계의 핵심입니다.
Q 40
정규화 · Softmax

다중 클래스 분류 출력층에서 각 클래스의 확률을 계산하는 Softmax 함수의 수식으로 올바른 것은?

  • ①
    $P(k) = \dfrac{x_k}{\sum_{j} x_j}$
  • ②
    $P(k) = \dfrac{1}{1 + e^{-x_k}}$
  • ③
    $P(k) = \dfrac{e^{x_k}}{\displaystyle\sum_{j=1}^{K} e^{x_j}}$

역전파 (Backpropagation) — 기울기 계산

단순 회귀 모델에서 MSE 손실의 가중치 $w$에 대한 기울기 $\dfrac{\partial L}{\partial w}$를 계산하시오.

◼ 조건
입력: $x = 3$, 가중치: $w = 0.5$, 바이어스: $b = 0$ (무시)
예측값: $\hat{y} = wx = 0.5 \times 3 = 1.5$
정답: $y = 3$
손실 함수: $L = \dfrac{1}{2}(\hat{y} - y)^2$ (MSE 단순화 형태)
  • ① $\dfrac{\partial L}{\partial w} = -4.5$
  • ② $\dfrac{\partial L}{\partial w} = -4.5$
  • ③ $\dfrac{\partial L}{\partial w} = 2.25$
  • ④ $\dfrac{\partial L}{\partial w} = -1.5$
Step 1. 손실 → 예측값 편미분 (∂L/∂ŷ):
$\dfrac{\partial L}{\partial \hat{y}} = \hat{y} - y = 1.5 - 3 = \mathbf{-1.5}$
Step 2. 예측값 → 가중치 편미분 (∂ŷ/∂w):
$\hat{y} = wx$ 이므로 $\dfrac{\partial \hat{y}}{\partial w} = x = \mathbf{3}$
Step 3. 연쇄 법칙(Chain Rule) 적용:
$\dfrac{\partial L}{\partial w} = \dfrac{\partial L}{\partial \hat{y}} \times \dfrac{\partial \hat{y}}{\partial w} = (-1.5)(3) = \mathbf{-4.5}$
Step 4. 가중치 업데이트 (lr=0.1):
$w_{\text{new}} = w - \alpha \dfrac{\partial L}{\partial w} = 0.5 - 0.1(-4.5) = 0.5 + 0.45 = \mathbf{0.95}$
→ 손실이 감소하는 방향으로 가중치가 업데이트됩니다.
정답: ① 또는 ② (동일값) $\dfrac{\partial L}{\partial w} = -4.5$
연쇄 법칙: $\dfrac{\partial L}{\partial w} = (\hat{y}-y) \cdot x = (-1.5)(3) = -4.5$
음수 기울기이므로 경사하강법으로 $w$를 증가시켜 손실을 줄이는 방향으로 업데이트합니다.
FP 4
역전파 + 가중치 업데이트 — Cross Entropy

이진 분류에서 Sigmoid + Binary Cross Entropy 손실을 사용할 때, 역전파의 최종 기울기 $\dfrac{\partial L}{\partial z}$ (z: Sigmoid 입력)는 놀랍도록 단순하다. 올바른 수식은?

◼ 조건 및 수식
Sigmoid: $\sigma(z) = \dfrac{1}{1+e^{-z}} = \hat{y}$
BCE 손실: $L = -[y\log\hat{y} + (1-y)\log(1-\hat{y})]$
  • ① $\dfrac{\partial L}{\partial z} = \hat{y}(1-\hat{y})$
  • ② $\dfrac{\partial L}{\partial z} = y - \hat{y}$
  • ③ $\dfrac{\partial L}{\partial z} = \hat{y}^2 - y$
  • ④ $\dfrac{\partial L}{\partial z} = \hat{y} - y$
정답: ④ $\dfrac{\partial L}{\partial z} = \hat{y} - y$

연쇄 법칙으로 전개: $\dfrac{\partial L}{\partial z} = \dfrac{\partial L}{\partial \hat{y}} \cdot \dfrac{\partial \hat{y}}{\partial z}$

$\dfrac{\partial L}{\partial \hat{y}} = -\dfrac{y}{\hat{y}} + \dfrac{1-y}{1-\hat{y}}$, $\quad \dfrac{\partial \hat{y}}{\partial z} = \hat{y}(1-\hat{y})$ (Sigmoid 미분)

두 항을 곱하면 놀랍게도 깔끔히 소거되어: $\dfrac{\partial L}{\partial z} = \hat{y} - y$

이것이 딥러닝에서 Sigmoid+BCE 조합이 선호되는 이유 중 하나입니다 — 역전파 기울기가 "예측값 − 실제값"으로 극도로 단순해집니다.

TC 3
K-Fold 교차 검증 — 훈련/검증 비율

총 1,000개의 데이터를 5-Fold 교차 검증으로 학습할 때, 각 Fold에서 훈련에 사용되는 샘플 수는?

  • ① 200개
  • ② 800개
  • ③ 500개
  • ④ 1,000개
정답: ②
5-Fold에서 데이터를 5개 동일 구간으로 분할 → 각 구간: 1,000/5 = 200개
각 Fold에서 검증 세트 = 1개 구간 = 200개
훈련 세트 = 나머지 4개 구간 = 800개
이 과정을 5번 반복 → 모든 데이터가 1번씩 검증에 사용됩니다.
TC 4
Dropout — 추론 시 출력 스케일링

Dropout 비율 $p = 0.4$로 학습된 레이어가 있다. 훈련 시 뉴런 출력이 $h = 5.0$이었다면, 추론(Inference) 시 적용되는 스케일링 후 기댓값은?

◼ Dropout 스케일링 원리
훈련 시: 확률 p로 뉴런 비활성화 → 기대 출력 = $h(1-p)$
추론 시 (Inverted Dropout): 가중치에 $(1-p)$ 곱함
추론 시 보정 출력: $h_{inf} = h \times (1-p)$
  • ① 2.0
  • ② 3.0
  • ③ 4.0
  • ④ 5.0
정답: ②
$h_{inf} = h \times (1 - p) = 5.0 \times (1 - 0.4) = 5.0 \times 0.6 = \mathbf{3.0}$
훈련 시에는 40% 뉴런이 꺼지므로 살아있는 뉴런의 평균 기여를 추론에 맞추기 위해 $(1-p)=0.6$을 곱합니다. ④는 스케일링 없이 사용하면 출력이 훈련 대비 과대 예측되는 오류가 발생합니다.
TC 5
경사하강법 — 가중치 업데이트 수치 계산

SGD를 사용하여 아래 조건으로 가중치를 2회 업데이트할 때, 최종 가중치 $w$의 값은?

◼ 조건
초기 가중치: $w_0 = 1.0$
학습률: $\alpha = 0.1$
1회차 기울기: $\dfrac{\partial L}{\partial w}\bigg|_1 = 2.0$
2회차 기울기: $\dfrac{\partial L}{\partial w}\bigg|_2 = -1.5$
업데이트 규칙: $w_{t+1} = w_t - \alpha \cdot \dfrac{\partial L}{\partial w}$
  • ① $w = 0.65$
  • ② $w = 0.75$
  • ③ $w = 0.95$
  • ④ $w = 1.15$
1회차 업데이트:
$w_1 = w_0 - \alpha \cdot g_1 = 1.0 - 0.1 \times 2.0 = 1.0 - 0.2 = \mathbf{0.8}$
2회차 업데이트:
$w_2 = w_1 - \alpha \cdot g_2 = 0.8 - 0.1 \times (-1.5) = 0.8 + 0.15 = \mathbf{0.95}$
→ 기울기가 음수이면 가중치를 증가 방향으로 업데이트합니다.
정답: ③ $w = 0.95$
1회 업데이트: $1.0 - 0.1(2.0) = 0.8$
2회 업데이트: $0.8 - 0.1(-1.5) = 0.8 + 0.15 = 0.95$
음수 기울기는 손실을 낮추기 위해 가중치를 증가시키는 방향입니다.
